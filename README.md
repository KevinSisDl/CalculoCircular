## Programa de cálculos de círculo en Python
Este programa en Python permite calcular el perímetro, área y diámetro de un círculo dado a su radio. Proporciona una interfaz de usuario sencilla y fácil de usar para que el usuario pueda seleccionar la opción deseada.

## Características
El programa verifica que el radio ingresado por el usuario sea un número mayor que cero, realiza los cálculos de perímetro, área y radio utilizando la constante ´pi` con una precisión de 4 decimales (3.1416), presenta un menú claro y fácil de usar proporcionando opciones numeradas para que el usuario pueda seleccionar la acción deseada  y cuenta con una animación de carga que muestras puntos suspensivos mientras realiza los cálculos para indicar que está procesando la información

## Requerimientos 
Python 3.x

## Estructura del código 
El programa consta de dos componentes principales:
  - La clase ´Círculo`, que representa un círculo y proporciona métodos para calcular su perímetro , área y diámetro 
  - La función ´menu_calculos´, que proporciona una interfaz de usuario y interactúa con la clase ´Circulo´ para realizar cálculos

## Uso
  1. Ejecuta el programa
  2. Ingresa el radio del círculo cuando se te solicite
  3. Selecciona la opción deseada del menú (calcular perímetro, área o diámetro)
  4. observa los resultados de los cálculos 
  5. Al seleccionar la opción “Salir del programa”, el programa se cerrará

## Notas
  - Asegurate de ingresar un radio valido mayor que cero para obtener resultados precisos
  - El programa utiliza la constante π con un valor predeterminado de 3.1416, pero puede ser ajustado según tus necesidades
